# Coffers

A modern, from-scratch economy plugin for Paper with built-in Vault compatibility and a clean developer API.

## Modules

- `coffers-paper`: the main plugin server owners install
- `coffers-api`: the optional developer API jar

## Downloads

- Server owners: use `release/Coffers.jar`
- Developers: optional `release/Coffers-API.jar`
